# \# Can You Escape?

# 

# > Psychological Horror Puzzle Game for Steam (Windows)

# 

# \---

# 

# \# Project Overview

# 

# \*\*Can You Escape?\*\* is a premium single-player 2D psychological horror puzzle game developed with Unity 6.

# 

# The game combines environmental storytelling, inventory-based puzzles, nonlinear decision making and psychological horror into a cinematic experience designed specifically for Steam.

# 

# The project targets English-speaking audiences in Tier-1 countries, including the United States, Canada, United Kingdom, Australia and New Zealand.

# 

# The game is developed exclusively in English.
